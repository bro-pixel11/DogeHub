# DogeHub

Multi-game Roblox hub made by bro-pixel11.

## Features

- Modular hub system
- Automatic game detection
- GitHub loader support
- Word Bomb support
- Fast loader architecture
- Expandable game modules

## Supported Games

- Word Bomb

## Loader

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/bro-pixel11/DogeHub/main/loader.lua"))()
```

## Structure

```text
DogeHub/
├── loader.lua
├── games/
│   ├── wordbomb.lua
│   └── universal.lua
```
